Pokedex App

Overview

This is a Pokedex App that I made that utilizes JavaScript, HTML and CSS. This is my first time working with JavaScript. The app retrieves information for the Pokemon from an API called PokeApi. Users are able to view a list of pokemon and are able to click to see more details about each pokemon. The dependencies are JQuery and Bootstrap and follows ESLint rules.

Purpose

The goal of this project was to create my first fully functioning JavaScript web application in my CareerFoundry bootcamp.

Objective

In this project, I aimed to build a small web application with HTML, CSS and JavaScript that loads data from an external API and enables the viewing of data points in detail.

Project Duration

One month

Tools Used

  • HTML
  • CSS
  • JavaScript

Process

I began this project learning all of the basics of JavaScript, like primitive and complex data types, conditionals and loops, functions, parameters and objects. I set up the basic structure of the application using HTML and CSS then added JavaScript. I followed the project brief's instruction on what the User Interface should look like. After laying the groundwork for the site, I introduced the PokeApi to my app. I finished the project by implementing Bootstrap to create the navbar, buttons and modal.

Challenges

I struggled with the design aspect of this project while using Bootstrap. To solve my issues, I asked my tutor for guidance, as well as continuing to read about the framework on it's website and learn from the experience of other developers.

Results

This project set a strong foundation for my understanding of JavaScript. I have revisted the project and assignment several times as refreshers while I worked on new projects.
