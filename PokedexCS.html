Pokedex App

OVERVIEW

This is a Pokedex App that I made that utilizes JavaScript, HTML and CSS. This is my first time working with JavaScript. The app retrieves information for the Pokemon from an API called PokeApi. Users are able to view a list of pokemon and are able to click to see more details about each pokemon. The dependencies are JQuery and Bootstrap and follows ESLint rules.
