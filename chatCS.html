Chat App

OVERVIEW

Chat App was developed using React Native, Expo and GiftedChat to create a messaging application. The app provides users with a chat interface with options to share images and their location. Users can choose between four different background colors for their chat which helps the app feel more personalized. This app was styled according to the given screen design. The users' conversations are stored in Google Firestore Database and the users can retrieve their messages without internet connection.
